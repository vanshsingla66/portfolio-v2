# Vansh — Portfolio

A dark, premium personal portfolio for **Vansh** — Full Stack Developer & AI/ML enthusiast.

Built from scratch with a focus on motion, glassmorphism, and an "AI engineer building future products" aesthetic.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** (build tooling)
- **Framer Motion** (animations — scroll reveals, magnetic buttons, tilt cards, rotating text)
- **react-icons** (tech & social icons)
- Hand-crafted CSS design system (deep-black theme, electric blue/purple glow, glass panels)

## Sections

Hero · About · Tech Stack · Projects · Journey · GitHub · Contact

## Develop

```bash
npm install
npm run dev      # start dev server
npm run build    # type-check + production build
npm run preview  # preview the production build
npm run lint     # lint
```

## Customize

All content lives in [`src/data/profile.ts`](src/data/profile.ts) — edit name, links,
email, tech stack, projects, and journey there. Design tokens (colors, radius,
spacing) live at the top of [`src/index.css`](src/index.css).
